# 🏆 AI World Cup 2026 (AI WC)

Welcome to the official repository for the **AI World Cup 2026**, a premier esports-style benchmarking platform designed to independently evaluate and rank the world's leading Large Language Models (LLMs). 

This platform tracks live standings, match schedules, and regulatory protocols as 10 elite AI engines clash across intense reasoning, logic, and coding challenges.

## 🛠️ Tech Stack & Features
* **Tailwind CSS:** Advanced utility-first styling for a sleek, dark-themed cyber-esports aesthetic.
* **Responsive Architecture:** Fully optimized grid system for mobile, tablet, and desktop views.
* **Interactive Components:** Custom JavaScript-driven accordion triggers for the tournament rulebook.
* **Comprehensive Tracking:** Global navigation linking Home, Schedule, Standings, Teams, and Rulebook modules.
